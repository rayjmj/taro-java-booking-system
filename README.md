预约系统全栈项目

项目概述
这是一个全栈预约系统，包含 Spring Boot 后端和 Taro 微信小程序前端。支持手机号验证码登录、预约创建（含幂等性控制）、预约列表查看等功能。

快速开始
环境要求
后端：Java 21、MySQL 8.0+、Gradle 8.5
前端：Node.js 16+、微信开发者工具
数据库：MySQL（用户名：root，密码：123456）

#步骤一 启动后端服务
1.进入后端目录
cd booking-system-backend
2.运行
BookingApplication.java类名右键运行

#步骤二 启动小程序
1.进入前端目录
cd booking-system-frontend
2.安装依赖：
npm install
3.开发模式运行
npm run dev:weapp

#步骤三：导入微信开发者工具
1.打开微信开发者工具
2.选择"导入项目"
3.目录选择：booking-system-frontend/dist
4.AppID：使用测试号或自己的 AppID
5.点击"导入"

#配置服务器地址
// 在 src/config/api.ts 中修改 API 地址

#完整测试流程
步骤 1：登录系统
打开小程序，进入登录页
使用演示账号：
手机号：13800138000
验证码：123456（固定验证码）
点击登录，成功后跳转到首页

步骤 2：创建预约
点击"新建预约"或导航到创建预约页
填写表单：
服务名称：理发服务
预约日期：选择明天的日期
时间段：09:00-10:00
点击"创建预约"
验证幂等性：快速点击两次提交按钮，应只有第一次成功

步骤 3：查看预约列表
点击"我的预约"或从首页进入
查看刚创建的预约是否出现在列表中
验证预约信息是否正确显示

步骤 4：重复预约验证
再次创建相同服务的预约：
相同服务名称、相同日期、相同时段
应提示："该时间段已存在相同预约"
验证幂等性保护生效
